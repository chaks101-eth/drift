Wins

This is the moat. Booking.com structurally can't bundle because their teams own different inventory. Travelers save real time and money. Drift owns the post-purchase relationship.

Costs

Three API contracts (Duffel + Hotelbeds + activity provider). PCI compliance via Stripe Connect. Failure modes are gnarly — what if 7 succeed and the hotel rejects? Need careful retry + refund logic.

Dependencies

Duffel (flights, NDC), Hotelbeds or Booking.com Partners (hotels), GetYourGuide or Viator (activities), Stripe (charging), strong backend orchestration.
