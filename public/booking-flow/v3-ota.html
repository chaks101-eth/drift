Wins

Premium UX. Each booking is a self-contained moment with high craft (seat map, boarding pass). Users can book what they need, when they need it — including last-minute flights.

Costs

Doesn't differentiate from MakeMyTrip/Kayak structurally. You still need full transactional APIs (same as V2). User has to repeat steps for 8 items in a 5-day trip = friction.

Dependencies

Duffel for flights (with seat map data), Hotelbeds for hotels, Stripe for payment. Plus design time on the boarding pass + room picker craft moments.
